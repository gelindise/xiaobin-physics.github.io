弹簧惯性实验 — 受迫振动

相同弹簧、相同振动源，质量不同 → 惯性不同 → 振幅不同

固有频率 重物 f₀ = 0.56 Hz  |  轻物 f₀ = 1.13 Hz
受迫振动 驱动力频率 f 接近固有频率时，振幅最大（共振）
惯性原理 质量 ↑ → 惯性 ↑ → 受迫振幅 ↓ — 重物更难被驱动

实验说明

  • 两根弹簧完全相同（劲度系数 k=50 N/m），振动源在顶部同步驱动
  • 左：4kg 重物（橙色） 右：1kg 轻物（蓝色）
  • 点击"开始振动"，振动源以相同频率上下振动
  • 重物惯性大 → 难以被驱动 → 振幅很小
  • 轻物惯性小 → 容易被驱动 → 振幅很大
  • 拖动"振动频率"滑块，观察不同频率下的响应差异

物理原理

相同弹簧（k 相同）、相同驱动力频率 f，只有质量不同：

固有频率 f₀ = (1/2π) · √(k/m)

质量越大 → 固有频率越低 → 驱动频率远高于固有频率 → 重物跟不上振动 → 振幅小。

这正是惯性的直观体现：质量大的物体运动状态更难改变。

  • 为什么重物总是"慢半拍"？
  • 如果两个物体质量相等，振幅会怎样？
